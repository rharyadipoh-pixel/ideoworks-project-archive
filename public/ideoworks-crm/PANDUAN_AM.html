Alur Stage Deal

Lead (5%) → Qualified (15%) → Credentials Sent (20%) → Rate Card / Proposal Sent (35%) → Meeting Done (50%) → Hot Leads (65%) → Negotiation (80%) → Won (100%) / Lost (0%)

Angka dalam kurung = probabilitas closing. Weighted value di dashboard dihitung otomatis.

4 Alur Data

Kamu isi Form → Data masuk ke Form Responses → Script otomatis routing:
    Form 1 → tab Activities
    Form 2 → tab Accounts
    Form 3 → tab Deals (update baris existing)

Dashboard & Monday Review → terbaca otomatis ✓

5 Dashboard & Monday Review

■ Dashboard

  • Total pipeline per AM
  • Weighted value vs target bulan / kuartal
  • Breakdown per stage
  • Deals stale (>14 hari tidak ada aktivitas)
  • Win rate & deal count

■ Monday Review

  • Semua open deals terdaftar
  • Last activity date per deal
  • Next action & due date
  • Flag deals yang sudah overdue
  • Cocok dibuka tiap Senin pagi

7 Tips Harian

  • Simpan 3 link form di HP — pakai shortcut browser atau pin di WA saved messages supaya mudah diakses kapan saja
  • Isi Form 1 setiap habis aktivitas — jangan tunggu numpuk, biar data dashboard akurat dan pipeline termonitor dengan baik
  • Update Form 3 tiap ada perubahan stage — ini yang menggerakkan angka pipeline dan weighted value di dashboard
  • Cek Monday Review tiap Senin pagi — lihat next action yang sudah jatuh tempo, prioritaskan follow-up hari itu
