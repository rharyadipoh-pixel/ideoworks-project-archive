Revenue Tracking Per Deal

What to capture in CRM for each deal

1
Deal Value (Expected Revenue)
Total contract value in USD. E.g. $48,000/year retainer. Set as soon as qualified.
2
Probability %
Weighted by stage. Stage 3 = 35%, Stage 5 = 70%. Drives forecast accuracy.
3
Expected Close Date
Used for pipeline forecasting. Flag if overdue — triggers manager alert.
4
Source Attribution
Which campaign / channel originated this deal. Calculates true ROI per campaign.
5
Service Type
Strategy, Creative, Media, Analytics — for capacity planning.

Week 3 — Automation & Reporting

Connect everything and build the dashboard

1
Build Make.com automations
Build the 3 priority automations: Calendly→HubSpot, Form→Slack, Deal Stage→Task Assignment.
Make.com
2
Create Looker Studio dashboard
Connect GA4 + HubSpot + LinkedIn Ads + Google Ads. Build pages: Overview, Channel Performance, Pipeline Health, Revenue Forecast.
Looker Studio
3
Set up HubSpot email sequences
Create automated nurture sequences for: new leads (intro series), stalled deals (re-engage), and closed-lost (90-day follow-up).
HubSpot Workflows
W4

Week 4 — Launch & Optimize

Go live, test, and tune

1
Launch outbound sequences
Start Apollo sequences. Cap at 30–50 emails/day to warm up sending domain. Monitor open rates (target: 40%+) and reply rates (target: 5%+).
2
Test all integrations end-to-end
Submit a test form. Book a test Calendly. Verify HubSpot contact appears, deal created, Slack fires, task assigned.
3
Weekly pipeline review ritual
Every Monday: review dashboard. Update deal stages. Flag stuck deals. Adjust budget allocation based on CPL by channel.
4
Establish 30/60/90 targets
30 days: 10 qualified leads. 60 days: 3 proposals sent. 90 days: 1 closed deal. Adjust monthly based on conversion data.

The "Campaign Health" Quick Check

Ask these 3 questions every Monday to diagnose problems fast

❓
"Are enough leads entering the pipeline?"
If no → fix the top of funnel (ad spend, outbound volume, website conversion rate)
❓
"Where are deals getting stuck?"
Check stage-by-stage conversion. Stalled at Proposal → improve deck. Stalled at Negotiation → pricing issue.
❓
"Which channel has the best ROI?"
Compare CAC by source. Double down on the channel that closes deals, not just generates leads.
